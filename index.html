  • Explicando tag p

A tag 'p' define um parágrafo.

  • Explicando tag mark

A tag 'mark' define texto que deve ser marcado ou destacado.

  • Explicando tag small

A tag 'small' define texto menor (como direitos autorais e outros comentários paralelos).

Dica: Esta tag não está obsoleta, mas é possível obter mais (ou o mesmo) efeito com CSS.

  • Explicando tag i

A tag 'i' define uma parte do texto em uma voz ou humor alternativo. O conteúdo interno é normalmente exibido em itálico.

Exemplo da tag i.

  • Explicando tag u

A tag 'u' representa algum texto que é desarticulado e estilizado de forma diferente de texto normal. O conteúdo interno geralmente é exibido com um sublinhado.

Exemplo de uso da tag u

  • Explicando tag strong

A tag 'strong' é usado para definir texto com grande importância. O conteúdo interno é normalmente exibido em negrito.

Exemplo de uso da tag strong

  • Explicando tag ol

A tag 'ol' define uma lista ordenada. Uma lista ordenada pode ser numérica ou alfabética.

  • Explicando tag ul

A tag 'ul' define uma lista não ordenada (marcada).

Exemplo da tag ul:

  • Lista não ordenada
  • Lista não ordenada

  • Explicando tag li

A tag 'li' define um item de lista.é usado dentro de listas ordenadas(ol), listas não ordenadas (ul), e em listas de menus (menu).

  • Explicando tag a

A tag 'a' define um hiperlink, que é usado para vincular de uma página para outra. O atributo mais importante do o elemento é o href atributo, que indica o destino do link.

Exemplo da tag a: Visite a página w3schools

  • Explicando tag hr

A tag 'hr' define uma quebra temática em uma página HTML (por exemplo, uma mudança de tópico). Exemplo abaixo

  • Explicando tag sub

A tag 'sub' define texto subscrito. O texto subscrito aparece metade de um caractere abaixo do linha normal.

Exemplo: H2O

  • Explicando tag sup

A tag 'sup' define texto sobrescrito. O texto sobrescrito aparece metade de um caractere acima do linha normal.

Exemplo: WWW[1]

  • Explicando tag blockquote

A tag 'blockquote' especifica uma seção que é citada de outra fonte.

Exemplo de uso da tag blockquote.

  • Explicando tag font

A tag 'font' foi usado em HTML 4 para especificar a face da fonte, tamanho da fonte e cor do texto. Não suportado em HTML5

  • Explicando tag del

A tag 'del' define o texto que foi excluído de um documento. Navegadores normalmente irá atacar uma linha através de texto excluído.

Exemplo: minha cor favorita é azul vermelho

  • Explicando tag ABBR

A tag 'ABBR' define uma abreviação ou um acrônimo, como "HTML" "CSS", "Sr.", "Dr.", "ASAP", "ATM".

Exemplo: O Real Madrid (RMA) foi o grande campeão.
